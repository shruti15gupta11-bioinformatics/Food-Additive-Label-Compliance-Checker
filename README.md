# Food Additive & Label Compliance Checker

Python-based project for identifying food additives from ingredient labels and generating a compliance report.

## Features

- Ingredient label analysis
- Food additive detection
- Compliance reporting
- File-based input handling

## Technologies

- Python
- Food Safety Analysis

## Example Output

```text
Additive Found: Sodium Benzoate
Function: Preservative

Additive Found: Tartrazine
Function: Artificial Color
```
